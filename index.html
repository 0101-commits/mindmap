섹션 관리

마인드맵은 캔버스로 렌더링됩니다. 키보드 사용자는 상단 그룹 필터·단계 필터(Tab 이동, Enter/Space 토글, Shift+Enter 단독 보기)와 검색창을 이용해 탐색할 수 있습니다.

노드를 선택하세요

노드를 클릭하면 상세 설명과 연결관계가 나타납니다. 더블클릭 → 수정, 선 더블클릭 → 실선/점선 전환.

    연결된 노드

    저장 내역 (History)

    항목을 클릭하면 해당 시점으로 맵을 복원합니다.

    색상 커스터마이징

    대분류(브랜치)별 색상을 바꾸면 노드·연결선·상단 필터·범례에 즉시 반영됩니다.

    노드 클릭 → 강조·상세  ·  더블클릭 → 수정  ·  선 더블클릭 → 실선/점선  ·  칩 더블클릭 → 단독  ·  휠 확대 / 드래그 이동

    노드 추가

    선택한 노드와 새 노드를 선으로 연결합니다. (선을 더블클릭하면 실선/점선 전환)
