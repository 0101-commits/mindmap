# 개인 E2E 성과관리 마인드맵

순수 HTML/CSS/Vanilla JS 기반 인터랙티브 마인드맵. 빌드 과정 없이 `index.html`을 브라우저로 바로 열면 동작합니다 (vis-network는 CDN 로드).

## 구성
| 파일 | 역할 |
|------|------|
| `index.html` | UI 뼈대 + 라이브러리 로드 |
| `style.css` | 디자인 · 사이드 패널 |
| `script.js` | 마인드맵 렌더링 · 필터 · 패널 상호작용 |
| `data.js` | 노드/엣지 데이터 (각 노드 `group` 속성 포함) |

## 내용
HCG 성과관리 OKR Manual + PM Manual(Masterbook) + 기획회의(개인 E2E 성과관리·1단계 목표수립) 내용을 결합.

흐름: **목표수립 → 체크인 → 평가 → 피드백** / 핵심: **직무내용 → CSF 도출 → KPI 매칭**.

그룹: `프로세스 · 목표 · 맥락 · 지표 · 원칙 · 운영` (상단 필터로 그룹별 표시/숨김).

## 사용법
1. `index.html` 더블클릭 (인터넷 연결 필요 — CDN).
2. 노드 클릭 → 우측 상세 패널. 상단 칩 → 그룹 필터. 검색창에 키워드 후 Enter.
